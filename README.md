# hb-xcb

Harbour module implementing bindings to [XCB](https://en.wikipedia.org/wiki/XCB).

## Building

- Build static library:

   ```
   hbmk2 hbxcb.hbp
   ```

- Build and run sample and test code:

   ```
   cd cd examples/harbour
   hbmk2 example_00.prg
   ./example_00
   ```
